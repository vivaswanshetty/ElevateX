ElevateX – The Skill Market for the New Gen*

ElevateX is a gamified micro-skill marketplace that connects people who need quick help with those who can deliver — for fun, XP, or micro-payments. Built with the MERN stack (MongoDB, Express, React, Node.js).

 🌟 Features:

- 🏠 Home feed showing trending tasks  
- 👤 User dashboard with XP and badges  
- ⚡ Real-time chat between clients and helpers  
- 🏆 Weekly leaderboard  
- 🎯 Gamification: earn XP, level up, and unlock badges  
- 🔒 JWT-based authentication  
- 🧠 Scalable architecture ready for AI matching and payments


*⚙️ Tech Stack:*

*Frontend:* React, Tailwind CSS, Axios, Socket.io-client.

*Backend:* Node.js, Express.js, Socket.io, JWT Auth .

*Database:* MongoDB + Mongoose  .
*Hosting (optional):* Vercel (frontend), Render/Railway (backend), MongoDB Atlas (DB).
